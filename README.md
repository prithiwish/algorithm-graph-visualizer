# Algorithm & Graph Visualizer

## 🚀 Features

### 🔴 Sorting Algorithms

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

### 🌐 Graph Traversal

- BFS (Breadth First Search)
- DFS (Depth First Search)

### ⚙️ Additional Features

- Directed & Undirected Graph Toggle
- Pause / Resume Sorting
- Comparisons & Swaps Counter
- Time Complexity Display
- Traversal Order Output
- Interactive UI for better understanding

---

## 🛠 Tech Stack

- React (Vite)
- JavaScript
- HTML & CSS

---

## ▶️ Run Locally

```bash
git clone https://github.com/prithiwish/algorithm-graph-visualizer.git
cd algorithm-graph-visualizer
npm install
npm run dev
```
